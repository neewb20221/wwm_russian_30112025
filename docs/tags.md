# Теги, используемые в переводе

- `<Название|780|#C|15>`Тег-ссылка на объект или информацию.Вместо "Название" указывайте нужный текст (можно на русском).
- `{0int}`Переменная, подставляемая внутрь фразы. Пример: `Login for 7 Days ({0}/7)` — сюда подставляется число.
- `#G140%#E Тут текст #G1#E`Пример оформления текста цветом.`#G...#E` — тег смены цвета (например, зелёного) для выделения числа, значения и пр.
- `id_2`Обращение или вызов объекта по идентификатору.
- `{ACR_VALUE}, {DOG_VALUE}, {month} {day}`Переменные, в которые будут подставлены значения. Аналогично шаблонам вида `{0}`, `{1}`, но из глобальных переменных.
- `$S $E` Предположительно... Условие аналог if else
